line() — Draw straight line segment

  line(x1, y1, x2, y2);

See also

  • Functions to draw figures
  • stroke() to set the line color
  • strokeWeight() to set the line width
  • noStroke() to set figure outlines invisible
  • strokeCap() to set the shape of line ending

draw() — The main animation loop

draw() is a function that the user should define in their program to enable animation. 
  function draw() {
    // Draw something here, for example:
    rect(10, 10, 100, 100);
  }
